# ABSTRACT

## 使用 Hugging Face 打造員瑛式思考生成器

本專案探索了如何使用 Hugging Face Inference API 構建一個基於人工智慧的正向思維生成系統。專案以「員瑛式思考」(Lucky Vicky)為核心概念,開發了一個能夠將負面事件重新詮釋為正向經歷的智能應用程式。

### 技術實現

專案整合了多個先進的大型語言模型(LLM),包括 Meta Llama 3.2 系列和 Microsoft Phi-3,透過 Hugging Face Inference API 提供服務。系統支援本地 Windows 環境和 Jupyter Notebook 兩種運行模式,並實現了完整的 Token 認證機制和錯誤處理流程。為提升使用者體驗,專案採用 Gradio 框架開發了互動式網頁介面,具備模型選擇、即時生成和結果展示等功能。

### 主要成果

本專案成功實現了:(1) 完整的 Hugging Face API 整合方案,包含詳細的配置文件和使用文檔;(2) 支援多提供商的彈性架構,可在 Hugging Face、Groq 和 OpenAI 之間切換;(3) 用戶友善的 Web 應用程式,提供直覺的操作介面和豐富的示例;(4) 完善的錯誤處理和故障排除機制,確保系統穩定性。

### 應用價值

本系統展示了如何將先進的 AI 技術應用於心理健康和正向思維領域,為使用者提供了一個創新的工具來重新框架負面經歷。專案的模組化設計和詳盡文檔使其具有良好的可擴展性和教學價值,適合作為 AI 應用開發的學習範例。
